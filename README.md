# Sistema-de-Pedidos-Sabor-Caribe-o-.
Este archivo HTML representa la estructura del sistema web propuesto en el Documento Visión del proyecto "Sabor Caribeño". 

Se incluyen las principales secciones funcionales del sistema, como el inicio de sesión del administrador, el panel de resumen, la gestión de pedidos por mesa, la visualización de productos y el registro de ventas diarias.

La estructura fue desarrollada utilizando HTML5 semántico, con el objetivo de organizar correctamente el contenido del sistema antes de aplicar estilos o funcionalidades dinámicas, permitiendo visualizar de manera clara la distribución de la información dentro de la aplicación web.
